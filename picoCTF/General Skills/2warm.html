Description

Can you convert the number 42 (base 10) to binary (base 2)?

Solution

Option 1:

Solve by basic math by power of 2
Just remember this for decimal to binary conversion

Bit Position Power of 2 Value Binary Digit Weight
02^010000 0001
12^120000 0010
22^240000 0100
32^380000 1000
42^4160001 0000
52^5320010 0000
62^6640100 0000
72^71281000 0000

Now as we want 42,

42 = 32 + 8 + 2
101010
